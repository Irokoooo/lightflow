# LightFlow 上架物料草案

## 基本信息

- 插件名称：流阅 LightFlow
- 版本号：0.1.0
- 插件形态：飞书多维表格侧边栏插件
- 可见范围：内部企业成员

## 一句话介绍

面向多维表格标注场景的侧边工作台，帮助 Contributor、Reviewer、QC 在同一侧边栏内完成阅读、聚合对比、翻译和填写。

## 详细介绍

在飞书多维表格里做 CB / Reviewer / QC 协作时，常见痛点不是“没有字段”，而是阅读、切换、对照、翻译、填写都被拆散在表格的不同位置：同题记录要来回找、长文本回复不易读、译文和评论要反复复制、待做项也缺少连续滑动处理的工作流。LightFlow 的目标就是把这条作业链路收回到侧边栏里，让作业人员围绕当前记录直接完成阅读、比对、翻译和填写。

当前版本支持：

1. 多视图工作台：可按角色切换 `reviewer`、`质检视图`、`速读翻译` 等视图。
2. 同 Prompt ID 聚合：在 Reviewer / QC 视图内自动聚合同题记录。
3. 字段翻译：支持自动翻译与手动翻译，适合多语言标注作业。
4. Markdown / LaTeX 渲染：提升长文本、回复和公式内容的可读性。
5. 布局编辑：支持字段拖拽排序、显示隐藏、翻译开关配置。
6. 仪表盘：支持待做统计与跳转下一条待做记录。

## 适用场景

- CB / Reviewer / QC 多角色协作标注
- 同题多记录的横向比对
- 多语言回复阅读与辅助翻译
- 多字段长文本内容的集中审核

## 核心卖点

1. 聚合阅读：把同一 `Prompt ID` 的关联记录组织到一个页面中。
2. 少切换：减少在表格行之间来回跳转。
3. 少复制：译文、回复、评论和填写区域在同一工作区完成。
4. 可配置：字段顺序、可见性、翻译开关都可按视图保存。

## 数据安全说明

默认走飞书 translation API（瞬时限流 20 QPS / 租户，基础免费版月总量 10,000 次），失败时回退到 MyMemory。MyMemory 走 HTTPS 但数据会离开飞书域，在插件设置里会明示风险并允许用户关闭。

## 当前已知限制

1. 大表场景下，聚合视图仍会受到表规模影响。
2. `jp.json` 当前先与英文文案保持一致，正式日语翻译计划在 `v0.2` 完成。
3. 手册助手为后续规划能力，当前版本暂不可用。

## 使用方法（3 步上手）

1. 打开飞书多维表格，点右上角「插件」→「LightFlow」
2. 顶部视图栏切换到对应角色视图（reviewer / 质检 / 速读）
3. 用 J/K 滑动浏览，N 跳到下一条待做
